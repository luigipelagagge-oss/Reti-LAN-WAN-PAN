Lab: Comunicazione Seriale (Ping-Pong)

Simuliamo una rete minimale tra due Arduino collegati via cavo (TX-RX).

Schema logico: Il TX di uno va all'RX dell'altro.

Il Protocollo:

  1. Arduino 1 invia "PING"
  2. Arduino 2 riceve, controlla e risponde "PONG"
  3. Arduino 1 riceve conferma.

Codice Arduino 1 (Trasmettitore)

void setup() { Serial.begin(9600); Serial.println("Inizio comunicazione..."); } void loop() { Serial.println("PING"); // Invio il pacchetto delay(1000); }
